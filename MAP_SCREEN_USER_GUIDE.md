# Map Screen User Guide

## üìç Map Screen Features (15 Key Points)

1. **Real-Time Location Tracking** - Shows your exact position on the map as you move
2. **Checkpoint Markers** - Displays all event checkpoints with green/blue markers
3. **Speed Monitor** - Shows current speed vs speed limit with warnings when overspeeding
4. **Checkpoint Counter** - Displays completed checkpoints (e.g., "3/10 checkpoints")
5. **Time Remaining** - Countdown timer showing time left to complete the event
6. **Route Tracking** - Blue line shows your traveled path during the event
7. **Automatic Checkpoint Detection** - Detects when you reach checkpoints automatically
8. **Offline Support** - Saves checkpoints locally when internet is unavailable
9. **Auto-Sync** - Syncs pending checkpoints to server when internet returns
10. **Voice Alerts** - Announces checkpoint completion, overspeeding, and time warnings
11. **Map Layers** - Switch between Normal, Satellite, Hybrid, and Terrain views
12. **SOS Emergency** - Quick call button to contact event organizer
13. **History Tab** - View all completed checkpoints with timestamps
14. **Screen Always On** - Phone screen stays awake during navigation
15. **Event Abort** - Option to abort event with secure code verification

## üó∫Ô∏è Location Buttons - When to Use What

### **Two Different Buttons:**

**1. üìç "My Location" Button (Bottom Tab - Left)**
- **Purpose:** Find and show your current location once
- **When to Use:** 
  - When you first open the map
  - When you're lost and want to see where you are
  - When you want to check your position without continuous tracking
- **What it Does:** Zooms to your current location but doesn't follow you as you move

**2. üéØ "Navigation/Following" Button (Floating Button - Bottom Right)**
- **Purpose:** Continuous real-time tracking with auto-follow
- **When to Use:**
  - During active navigation/driving
  - When you want the map to move automatically as you travel
  - During the event to track your progress continuously
- **What it Does:** 
  - Centers map on your location
  - Automatically follows you as you move
  - Keeps you centered on screen during navigation
  - **IMPORTANT:** Stops following if you manually zoom or pan the map

### **Simple Usage Flow:**

**Starting Navigation:**
1. Press "My Location" button ‚Üí Finds your position
2. Press "Navigation" button ‚Üí Starts continuous tracking
3. Map now follows you automatically as you drive

**During Navigation:**
- Map stays centered on your car icon
- Blue line shows your traveled path
- If you zoom/pan manually, auto-follow stops
- Press "Navigation" button again to resume following

**Quick Position Check:**
- Just tap "My Location" once
- No need for continuous tracking
- Map zooms to your location

**Best Practice:** Use "Navigation" button during events for hands-free navigation!
